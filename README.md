# Hasshin Labo

発信ログ、noteシリーズ、note記事候補をスマホでも管理できるWebアプリです。

公開URL:

```text
https://mcwgw408-oss.github.io/hasshin-Labo/
```

スマホでは、このURLをSafariやChromeで開けば操作できます。最後の `/` まで入れて開くと404になりにくいです。

## できること

- 今週のテーマとタスクを管理（「今週を締めて記録に残す」で週ごとの記録が貯まる）
- 投稿ログを記録（数値・振り返りは折りたたみで、あとから追記できる）
- noteシリーズを管理
- note記事候補を管理（検索と「公開済み・保留も表示」あり）
- 仮メモを記事候補・X候補・Threads候補に昇格できる
- SNS候補タブでXとThreadsを切り替えて管理
- データのバックアップ書き出し・読み込み（JSONファイル）
- スマホでは下部タブで画面を切り替え

## 404が出るとき

次を順番に確認してください。

1. URLが `https://mcwgw408-oss.github.io/hasshin-Labo/` になっているか確認する。
2. `hasshin-Labo` の大文字小文字が同じか確認する。
3. URLの最後に `/` を付ける。
4. ブラウザを更新する。

## データについて

入力したデータは、その端末のブラウザに保存されます。

PCで入力したデータはPCに、スマホで入力したデータはスマホに保存されます。自動同期はありませんが、トップ画面の「データを書き出す」でJSONファイルを保存し、別の端末で「バックアップを読み込む」をすると、手動で引っ越しできます。

**大切：** ブラウザの保存データは、端末の不調・データ削除・長期間開かないこと（特にiPhoneのSafari）で消えることがあります。週に1回くらい「データを書き出す」でバックアップを残しておくと安心です。
